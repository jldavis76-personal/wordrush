A fun reading practice app for kids ages 5-9, built with React, TypeScript, and Tailwind CSS.

## Features

- **Reading Race**: Timed reading comprehension for ages 8+ (measures WPM)
- **Word Catcher**: Sight word matching game for ages 5+ with text-to-speech
- **Avatar Shop**: Earn coins and unlock cool items for your avatar
- **Kid-Friendly Design**: Large buttons, bright colors, and celebration animations

## Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- No backend (V1 uses React state only)
